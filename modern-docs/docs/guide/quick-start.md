# Quick Start

Get the Tuva Data Exploration code base from GitHub first.

## Step 1: Import dependencies

Get the version of React from [reactjs.org](https://reactjs.org/docs/add-react-to-a-website.html) (15.6.1) for React, React-DOM and jquery [3.2.1](https://jquery.com/download/) (soon the jquery reference will be removed).

```html
<script src="../lib/jquery.js"></script>
<script src="../lib/react.js"></script>
<script src="../lib/react-dom.js"></script>
```

## Step 2: Import tuva-data-tools lib

Import the library and style file from the provided repo.

```html
<!-- Add your import code here -->
<link rel="stylesheet" href="path/to/styles.css">
<script src="path/to/tuva-data-tools.js"></script>
```

## Step 3: Initialize

Initialize the tool with your configuration.

```javascript
// Example initialization code
TuvaDataTools.init({
  // config
});
```
