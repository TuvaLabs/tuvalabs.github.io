    Tuva Listeners and Methods

    Various Tuva events and listeners:
    Methods:

    changeMode( mode ):  Chagne the Toolbar mode - Default, Elementry, Middle School, High School. Based on the criteria provided, it will simplify toolbar options to reduce nuber of actions user may take.

    changeTheme( theme ):  Chagne the foreground and background of the environment.

    resize():  Adjust the visualization environment based on the window resize, or post zoom action.

    getPlotState():  Get current plot state.

    setPlotState():  Set plot state.

    getMetaData():  Get metadata specific to current environment.
